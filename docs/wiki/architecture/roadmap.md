# Extension Builder — Roadmap

## MVP (4 semanas)

### Semana 1 — Base
- Frontend Chat IA (Astro + Tailwind)
- Módulo DSL
- Generación simple de extensiones

### Semana 2 — Validación y publicación
- Pipeline de validación automática
- GitHub API automation (Octokit)
- Endpoint backend para creación de repos

### Semana 3 — UX completa
- UI pulida del flujo completo
- Feedback de errores al usuario
- Integración end-to-end

### Semana 4 — Catálogo y persistencia
- Guardado de extensiones generadas
- Catálogo básico (listado, búsqueda)
- Historial por usuario

## Post-MVP

### Fase 2 — Validación avanzada
- Compilación simulada PXT
- Testing automático de bloques generados
- Corrección iterativa mejorada

### Fase 3 — Marketplace
- Sistema de rating y comentarios
- Fork/remix de extensiones
- Templates educativos
- Categorías y featured extensions

### Siguiente nivel
- Generación de juegos completos
- Editor visual híbrido (bloques + IA)
- Integración con plataforma de juegos propia
