# Plan: Semana 1 — MVP Base (Chat + DSL + Generación)

## 1. Objetivo
Implementar la base del proyecto: estructura monorepo, tipos DSL compartidos,
Edge Functions de traducción prompt→DSL y generación DSL→código, y frontend
Astro con chat UI.

## 2. Contexto actual
Proyecto nuevo. Wiki y planos listos. Sin código todavía.

## 3. Resultado esperado
- `frontend/` → Astro + Tailwind con chat UI funcional
- `supabase/` → Edge Functions: translate, generate, validate, publish
- Tipos DSL compartidos en TypeScript
- Pipeline mínimo: prompt → DSL → código PXT

## 4. Stack
- Frontend: Astro + Tailwind CSS + TypeScript
- Backend: Supabase Edge Functions (Deno)
- IA: OpenAI API
- GitHub: GitHub REST API (fetch, sin Octokit)

## 5. Arquitectura
```
frontend/ (Astro) → POST /functions/v1/translate → Edge Function → OpenAI → DSL YAML
                  → POST /functions/v1/generate  → Edge Function → OpenAI → código PXT
```

## 6. Cambios por archivo

### Scaffold
- `package.json` — root monorepo
- `frontend/package.json` — Astro project
- `supabase/` — Supabase config

### Tipos DSL (`supabase/shared/dsl-types.ts`)
- `DSLExtension` — root type
- `DSLFeature` — union of all feature types
- `FeatureShooting`, `FeatureEnemySpawner`, etc.

### Edge Functions (`supabase/functions/`)
- `translate/index.ts` — prompt → DSL YAML
- `generate/index.ts` — DSL → código
- `validate/index.ts` — check código generado
- `publish/index.ts` — crear repo GitHub

### Frontend
- `frontend/src/pages/index.astro` — chat UI
- `frontend/src/components/ChatBox.astro` — input + response

## 7. Orden de implementación
1. Scaffold monorepo + instalar dependencias
2. Tipos DSL compartidos
3. Edge Function: translate
4. Edge Function: generate
5. Frontend Astro + chat UI
6. Integrar frontend con Edge Functions
7. Ingestar a la wiki

## 8. Riesgos
- Supabase CLI no instalado → instalarlo vía npm
- OpenAI API key no disponible → placeholder + env var
- Sin proyecto Supabase real → funciones deployables después

- [ ] `npm run dev` levanta frontend Astro
- [ ] Chat UI acepta input y muestra respuesta
